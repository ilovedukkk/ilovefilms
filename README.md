# ilovefilms

Веб-приложение для просмотра каталога фильмов и сериалов с данными из TMDB, поиском раздач через Jackett-совместимые парсеры и воспроизведением через TorrServer.

## Возможности

- Главная страница с hero-блоком и несколькими подборками контента
- Разделы `Фильмы`, `Сериалы`, `Рекомендации`, `История`, `Буду смотреть`
- Детальная страница фильма или сериала с описанием, жанрами, актёрами и похожими тайтлами
- История просмотра и локальный список `Буду смотреть`
- Поиск по TMDB
- Поиск раздач через несколько Jackett-парсеров с fallback
- Фильтры в окне торрентов:
  - по названию
  - по качеству
  - по количеству сидов
  - по сортировке
- Встроенный плеер с:
  - HTML5 playback
  - iframe fallback
  - внешним открытием потока
  - копированием ссылки
  - fullscreen
  - mute / pause / restart
  - автоскрытием интерфейса
- Отдельная страница Jackett
- Поддержка TV-пультов и навигации стрелками
- Система расширений в стиле Lampa

## Как запустить

Так как это статический фронтенд, достаточно открыть [index.html](/D:/ilovefilms/index.html) в браузере. Для нормальной работы сетевых запросов лучше запускать через локальный HTTP-сервер.

Пример:

```powershell
cd D:\ilovefilms
python -m http.server 8080
```

После этого откройте:

```text
http://localhost:8080
```

## Что нужно для работы

### 1. TMDB

В проекте уже указан ключ TMDB в [app.js](/D:/ilovefilms/app.js). Если будете публиковать проект публично, лучше вынести ключ из клиентского кода.

### 2. TorrServer

В настройках нужно указать адрес TorrServer, например:

```text
http://127.0.0.1:8090
```

### 3. Jackett-парсер

По умолчанию поддерживаются:

- `https://jac.red`
- `https://ru.jacred.pro`
- `https://jr.maxvol.pro`

Основной парсер выбирается на отдельной странице `Jackett`.

## Сохранение данных

Приложение сохраняет данные в `localStorage` браузера:

- настройки приложения
- выбранный Jackett-парсер
- адрес TorrServer
- история просмотра
- список `Буду смотреть`
- фильтры торрент-поиска
- установленные расширения

## Управление

### Клавиатура и TV-пульт

- `Arrow keys` — навигация
- `Enter / OK` — выбрать
- `Escape / Backspace` — назад / закрыть окно

### Плеер

- `Space / OK` — play / pause
- `Left / Right` — перемотка
- `Up / Down` — громкость
- `M` — mute
- `F` — fullscreen
- `R` — перезапуск потока

## Расширения

Приложение поддерживает внешние JS-расширения через глобальный объект `Lampa`.

Расширения могут:

- добавлять категории
- добавлять парсеры
- добавлять кнопки на страницу фильма
- слушать события приложения

Подключение и управление расширениями доступны в настройках.

## Ограничения

- Приложение зависит от доступности внешних API и парсеров
- Возможны ограничения CORS в зависимости от среды запуска
- Встроенное воспроизведение зависит от формата потока и поддержки браузера
- Клиентский TMDB API key не подходит для безопасного production-развёртывания
